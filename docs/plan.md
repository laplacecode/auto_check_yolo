# YOLOv5 实时检测系统 - 项目规划

## 项目目标

开发一个基于 YOLOv5 的实时屏幕检测系统,支持屏幕区域录制和实时目标检测。

## 技术方案

### 方案 A: 前端集成方案 (all_in_one/)

**架构**: 前端直接运行 YOLO 检测,无需后端服务器


### 方案 B: 前后端分离方案 (src/)
前端推送视频到后端
后端缓存视频帧
YOLO 检测后生成完整的 30 FPS 视频流
返回给前端流畅播放
这是一个流式视频处理方案。让我实现：

# 能不能做到yolo输出到前端的视频清晰度和帧率与录屏获内播放的清晰度和帧率一致。主要是确保yolo能完整识别和输出，延时可以忽略
YOLO 输出视频的清晰度 = 录屏清晰度（无压缩损失）
帧率稳定 30 FPS（不要丢帧）
YOLO 完整识别每一帧（不跳帧）
延迟可以接受（2-3秒都可以）
